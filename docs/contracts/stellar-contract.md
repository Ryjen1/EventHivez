# Stellar Smart Contracts

EventHivez utilizes Soroban smart contracts on the Stellar network to ensure transparent and secure event management and ticket processing.

## Overview

The platform is powered by two primary contracts:

1.  **Event Registry**: Manages event metadata, organizer verification, and inventory tracking.
2.  **Ticket Payment**: Handles the financial lifecycle of tickets, including purchases, escrow, and refunds.

For detailed information on the contract architecture, build process, and deployment scripts, please refer to the [Contract README](../../contract/README.md).
